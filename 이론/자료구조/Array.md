# 배열(Array)
* 데이터를 순차적으로 메모리에 저장하는 선형 자료구조
* 배열 내 모든 원소가 같은 자료형이여야 하며, 배열의 크기는 고정으로 변경할 수 없음
* 시간복잡도
  * 탐색 : O(1)
  * 삽입/삭제 : O(n)
* 장점
  * index로 임의 접근(random access)이 가능하여 검색 성능이 좋음
  * 연속된 메모리 공간에 존재하여 순차 접근(sequential access)도 빠름
* 단점
  * 연속된 메모리 공간에 존재하기 때문에 삽입/삭제시 원소를 한 칸씩 밀거나 당겨 옮겨야 하므로 속도가 느림
  * 배열의 크기를 바꿀 수 없음. 크기를 변경하려면 새로운 배열 선언 후 원래 배열의 값을 복사해야 함
  * 데이터가 삭제되어도 해당 메모리 공간을 빈 공간으로 남겨 재사용할 수 없으므로 메모리 낭비가 발생  
  (메모리 사용이 비효율적임)

## Python
* Python은 배열을 지원하지 않음
* Python의 List는 동적 배열(Dynamic Array)로, 자유롭게 확장이 가능함. 또한 서로 다른 자료형을 원소로 가질 수 있음
* [Numpy 모듈](https://github.com/ImJunHong/TIL/blob/master/Python%203/Modules/numpy.md)을 사용하면 Python에서 Array 사용 가능
